# abfab.social

Tijdgeest — Modern leesgenootschap / boekenclub webapp.

## Stack

- **Framework**: Next.js 16 (App Router) — in `website/`
- **Deployment**: Vercel
- **Database**: Supabase (PostgreSQL + REST API)
- **Media**: Cloudinary (boekomslagen)
- **Components**: Radix Primitives (`@radix-ui/react-*`)
- **Styling**: Tailwind CSS v4 — brand tokens in `website/app/globals.css`
- **Taal**: TypeScript

## Development
- IDE: Visual Studio Code
- Dev server: `cd website && npm run dev`
